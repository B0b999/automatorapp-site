Прив’язка до додатку/сервера

Після входу ви можете надсилати токен користувача на свій сервер для синхронізації:

firebase.auth().currentUser.getIdToken().then(token => {
  fetch("https://your-server.example.com/api/link", {
    method:"POST",
    headers:{ "Content-Type":"application/json" },
    body: JSON.stringify({ idToken: token })
  });
});
